# YiShan English Learning App

A modern English vocabulary learning application.

## Features
- Vocabulary books (CET-4, CET-6, Gaokao, Zhongkao, IELTS, TOEFL)
- Spaced repetition learning
- Progress tracking
- Score prediction

## Tech Stack
- React + TypeScript
- Cloudflare Pages
- Cloudflare D1 Database
- Cloudflare Workers API
