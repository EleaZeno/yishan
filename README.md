# 忆闪 (YiShan) - 科学记忆系统

> 基于遗忘曲线的智能记忆系统，科学记忆，过目不忘

## 🎯 核心特性

### 📚 智能学习
- **遗忘曲线算法**：根据艾宾浩斯遗忘曲线科学安排复习
- **自适应难度**：根据掌握程度动态调整学习内容
- **多种学习模式**：闪卡、测试、练习等多种交互方式

### 📊 数据分析
- **实时统计**：词库总量、掌握进度、学习效率
- **可视化图表**：趋势图、分布图、柱状图
- **详细报告**：学习进度、标签分析、效率评估

### 🎓 诊断测试
- **词汇测试**：评估词汇掌握程度
- **语法测试**：检验语法理解能力
- **阅读理解**：提升阅读能力
- **听力测试**：训练听力技能
- **写作测试**：改进写作水平

### 📱 跨平台支持
- **Web 应用**：随时随地访问
- **PWA 应用**：可安装为桌面/移动应用
- **离线模式**：无网络也能学习
- **数据同步**：自动同步学习进度

## 💾 数据管理

### 导出数据
- 进入后台管理
- 点击"导出"按钮
- 获得 JSON 格式的完整备份

### 导入数据
- 进入词库页面
- 点击"导入"按钮
- 选择 JSON 文件

### 数据位置
- 所有数据存储在浏览器 IndexedDB
- 不同浏览器/设备数据独立
- 清除浏览器数据会丢失本地数据

## 🔐 隐私与安全

- ✅ 所有数据存储在本地浏览器
- ✅ 不收集个人信息
- ✅ 不上传学习数据
- ✅ 完全离线可用

## 🐛 常见问题

### Q: 数据会丢失吗？
A: 数据存储在浏览器 IndexedDB，清除浏览器数据会丢失。建议定期导出备份。

### Q: 可以在多个设备同步吗？
A: 目前不支持云同步，每个设备独立存储。可通过导出/导入实现数据转移。

### Q: 支持离线使用吗？
A: 支持！安装 PWA 应用后可完全离线使用。

### Q: 如何删除词汇？
A: 进入后台管理，找到词汇，点击删除按钮。
