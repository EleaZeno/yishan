# yishan v2.0 功能路线图

基于 [用户分析文档](./docs/feature-analysis.md) 的规划。

---

## Phase 1: 内容类型系统 (v2.1) ✅ 已完成

**目标**: 实现多内容类型支持，为不同记忆场景提供差异化策略

### 已完成 ✅
- [x] 类型定义 (`types.ts`)
- [x] 算法适配 (`algorithm.ts`)
- [x] 数据库 schema 支持
- [x] 添加单词界面支持内容类型选择
- [x] 错题类型标签（概念混淆/计算失误等）
- [x] 公式类型分类
- [x] 内容类型筛选器（Library 页面）
- [x] 内容类型图标和颜色区分

---

## Phase 2: 记忆进度可视化 (v2.2) ✅ 已完成

**目标**: 让用户直观感知记忆效果

### 已完成 ✅
- [x] 记忆进度看板组件 (MemoryDashboard)
  - 当日待复习/已完成数量
  - 记忆留存率
  - 认知负荷评分
- [x] 7日记忆留存率曲线
- [x] 薄弱知识点提示
- [x] Dashboard 集成
  - 内容类型分布图
  - 学习建议自动生成

---

## Phase 3: 碎片化记忆模式 (v2.3) ✅ 已完成

**目标**: 适配通勤/排队等碎片场景

### 已完成 ✅
- [x] QuickReview 组件
  - 快速10张卡片复习
  - 滑动/键盘交互
- [x] TTS 语音朗读（浏览器原生 SpeechSynthesis API）
- [x] 进度条显示

---

## Phase 4: 自适应难度 (v2.4) ✅ 已完成

**目标**: 基于用户表现自动调整记忆难度

### 已完成 ✅
- [x] 反应时间因子（算法层）
- [x] 内容类型间隔调整
- [x] UserProfiler 组件
  - 最佳学习时段分析
  - 疲劳度检测
  - 每日目标设置

---

## Phase 5: 云同步与多设备 (v2.5) ✅ 已完成

**目标**: 跨设备续记，无缝切换

### 已完成 ✅
- [x] Workers API 后端
- [x] D1 数据库 schema
- [x] IndexedDB + 云同步双写层
- [x] 增量同步机制
- [x] Workers 已部署到 https://yishan-api.15703377328.workers.dev

---

## Phase 6: 反馈与激励 (v2.6) ✅ 已完成

**目标**: 即时反馈强化记忆，轻量激励保持动力

### 已完成 ✅
- [x] FeedbackSystem 组件
  - 成就系统（8种成就）
  - 成就解锁 Toast
  - 成就墙展示
- [x] 即时反馈增强
  - 遗忘原因分析提示
  - 记忆稳定性提示

---

## Phase 7: 记忆联想库 (v2.7) ⏳ 待实现

**目标**: 用户自定义联想内容增强记忆

### 待实现
- [ ] 自定义联想字段
  - 图片上传（R2 存储）
  - 个人笔记
  - 联想口诀
- [ ] 联想内容优先展示
- [ ] 公共联想库（可选）
  - 社区共享联想内容
  - 点赞/评分系统

---

## Phase 8: 批量导入导出 (v2.8) ✅ 已完成

**目标**: 降低迁移成本，支持备份

### 已完成 ✅
- [x] BatchImport 组件
  - CSV/TSV 导入
  - 预览和校验
  - 导入模板下载
- [x] 内容类型支持导入

### 待实现
- [ ] Anki 卡片包导入 (.apkg)
- [ ] 完整数据导出
  - JSON 格式
  - CSV 格式
  - Anki 格式
